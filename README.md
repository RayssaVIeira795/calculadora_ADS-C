Este código C++ implementa uma calculadora de linha de comando com diversas operações matemáticas. Desenvolvido como atividade escolar com o auxílio do professor, o programa oferece funcionalidades como soma, subtração, multiplicação, divisão, potenciação, raiz quadrada e cálculo de fatorial. A interface amigável solicita o nome do usuário e apresenta um menu de operações disponíveis. O código inclui tratamento de erros, como divisão por zero e cálculo de raiz quadrada de números negativos. Para a operação de fatorial, o programa utiliza uma função específica que formata o resultado com separadores de milhar no padrão brasileiro, facilitando a leitura de números grandes. O objetivo principal foi aplicar os conhecimentos de programação em C++ para criar uma ferramenta útil e interativa, demonstrando o aprendizado em sala de aula.
